اُپن چت چیست؟

اُپن چت یک پروژه متن باز چت بر پایه ASP.Net Core، Angular، SignlarR و MySql است. Back-End این سیستم توسط Docker در سرور های ابری Liara.ir میزبانی شده است.

در نسخه دسکتاپ این پروژه از تکنولوژی Electron Js به همراه DotNet Core و Angular استفاده شده. مزیت استفاده از این روش پشتیبانی همزمان از سیستم عامل های ویندوز، لینوکس و مک، به حداقل رساندن زمان توسئه و استفاده از یک بیس کد برای سهولت مدیریت بروزرسانی ها در تمام سیستم عامل هامیباشد.

برای نرم افزار موبایل از flutter استفاده شده. مزیت استفاده از این تکنولوژی پشتیبانی همزمان از Android و iOS در حالی است که هر دو نرم‌افزار کاملا native، روان و کم حجم هستند. به حداقل رساندن زمان توسئه و استفاده از یک بیس کد برای سهولت مدیریت بروزرسانی ها در هر دو سیستم عامل از مزایای دیگر این تکنولوژی است.
